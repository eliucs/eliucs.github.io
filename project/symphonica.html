Symphonica.

Using machine learning with TensorFlow to generate classical music.

GithubDevpost

Submitted To McHacks 2017 (McGill University, Montreal)

Technologies Used Python, Flask, TensorFlow

We wanted to make this hackathon a learning experience, so we decided to learn about recurrent neural networks and TensorFlow.

Symphonica sets up a recurrent neural network that we fed in over 600 midi files of classical music, ranging from Bach to Tchaikovsky. The midi files are broken down based off of their chords, and classified by chord structure. After 15 or so epochs and 4 hours later, the program generated decent sounding piano music!
